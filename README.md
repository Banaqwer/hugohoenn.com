## About

I am Hugo Hoenn, a self-taught developer with a global upbringing across Senegal, Shanghai, Europe, and the United States. Coding has been my constant since childhood, and over the years I have turned that curiosity into a habit of designing, building, and shipping real software.

The project featured on the site is the most ambitious side project I have undertaken to date. It demonstrates the full software-engineering lifecycle: clean modular architecture, validated data pipelines, automated testing, real-time observability, broker API integration, scheduled execution, and a live monitoring dashboard.

## Project highlights

- 30+ Python modules organised around a single shared core
- 15,000+ lines of production-quality code
- 32 financial assets monitored in real time
- 15+ years of historical data used for backtesting and walk-forward validation
- Three independent layers of validation: walk-forward testing, Monte Carlo simulation, and ablation studies
- Real-time Flask dashboard for live operations monitoring
- Full audit logging, configuration as code, and defensive failure handling
